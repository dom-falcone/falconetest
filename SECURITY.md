# Security Policy

## Supported Versions

Currently, only the latest version of the website hosted on the main domain is supported.

| Version | Supported          |
| ------- | ------------------ |
| Live    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it **confidentially** via email:

ðŸ“§ **dom.falcone.official@gmail.com**

### What We Expect:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

### What You Can Expect:
- Acknowledgment within 48 hours
- Status update within 7 days
- Notification when the issue is resolved

### Please Do Not:
- Disclose the vulnerability publicly before it is fixed
- Access or modify other users' data
- Perform automated scans that could disrupt the service

## Scope

### In Scope:
- Website and all subpages
- API endpoints
- Authentication / login system
- User data handling

### Out of Scope:
- Third-party services (e.g., Cloudflare, external APIs)
- Social engineering attacks
- Denial-of-service (DoS) attacks
- Physical security

## Acknowledgments

We appreciate responsible disclosure and will mention researchers who report vulnerabilities in our Hall of Fame (optional, upon request).
